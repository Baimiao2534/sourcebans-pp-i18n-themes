# Темы i18n для SourceBans++

Пакеты тем интернационализации (i18n) для веб-панели [SourceBans++](https://github.com/sbpp/sourcebans-pp) 2.0.0+. Каждая тема представляет собой полностью локализованную замену темы по умолчанию, переведённую на уровне шаблонов Smarty без изменения каких-либо PHP-файлов.

Этот проект является неофициальным расширением языковой поддержки, созданным сообществом для администраторов серверов SourceMod, которым нужна локализованная панель управления.

## Доступные темы

| Код | Язык | Название темы |
|-----|------|----------------|
| `en_US` | Английский (США) | SourceBans++ English (US) |
| `en_GB` | Английский (Великобритания) | SourceBans++ English (UK) |
| `zh_CN` | Упрощённый китайский | SourceBans++ 简体中文 |
| `zh_HK` | Традиционный китайский (Гонконг) | SourceBans++ 繁體中文（香港） |
| `zh_MO` | Традиционный китайский (Макао) | SourceBans++ 繁體中文（澳門） |
| `zh_TW` | Традиционный китайский (Тайвань) | SourceBans++ 繁體中文（台灣） |
| `ja_JP` | Японский | SourceBans++ 日本語 |
| `de_DE` | Немецкий | SourceBans++ Deutsch |
| `fr_FR` | Французский | SourceBans++ Français |
| `ru_RU` | Русский | SourceBans++ Русский |

В будущих версиях будут добавлены новые языки.

## Установка

1. Скачайте последний [ZIP-архив релиза](https://github.com/Baimiao2534/sourcebans-pp-i18n-themes/releases).
2. Распакуйте архив. Вы получите папку для каждого языка (например, `ru_RU`, `zh_CN`).
3. Загрузите нужные папки в каталог `themes/` вашей установки SourceBans++.
4. Войдите в панель администратора, перейдите в **Настройки → Темы** и выберите нужную языковую тему.

## Особенности

- Каждая тема содержит 93 файла, полностью повторяя структуру темы по умолчанию.
- Все видимые пользователю строки переведены с помощью условных блоков Smarty `{if/elseif}` на уровне шаблонов. Изменения PHP не требуются.
- Общие ресурсы (JS, CSS, шрифты, изображения) идентичны теме по умолчанию.
- Терминология согласована внутри одной языковой семьи (например, упрощённый и традиционный китайский используют единую терминологию с региональными адаптациями).

## Совместимость

- Веб-панель SourceBans++ 2.0.0 (и более поздние версии 2.0.x).
- PHP 8.5+ (требование SourceBans++ 2.0.0).
- Smarty 5 (поставляется с SourceBans++ 2.0.0).
- Веб-сервер с поддержкой UTF-8 (Apache или Nginx).
